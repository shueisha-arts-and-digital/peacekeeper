Peacekeeper

Peacekeeper helps you visually distinguish tabs by URL patterns, making it easier to identify different environments at a glance.

Getting Started

Configure your websites by editing the JSON array below. Each configuration object can contain:

  • name: A friendly name for the website
  • url: The URL pattern to match (supports regex)
  • favicon: Custom favicon configuration (optional)
  • style: Custom CSS string to apply (optional)

Basic Options

name (string, required)
A descriptive name for the website. This helps you identify different configurations.
url (string, required)
The URL pattern to match. Supports regex patterns for flexible matching.
Examples:
  • "^https?://localhost" - Matches localhost with http or https
  • "^https://.*\\.example\\.com" - Matches all subdomains of example.com
  • "^https://example\\.com/(staging|development)" - Matches specific paths
favicon (object, optional)
Custom favicon configuration for this website. Set to false to keep the original favicon.
Example: 
"favicon": {
  "rel": "icon",
  "type": "image/svg+xml",
  "href": "data:image/svg+xml,%3Csvg xmlns='http://www.w3.org/2000/svg' viewBox='0 0 16 16' width='16' height='16'%3E%3Cpath d='M8 1 L15 14 L1 14 Z' fill='%23FFC107' stroke='%23F57F17' stroke-width='0.5'/%3E%3C/svg%3E"
}
style (string, optional)
Custom CSS string to apply to the page. Useful for color-coding different environments.
Example: 
"style": "#app > div > div > div > section { background-color: rgba(253, 129, 122, 1) !important; }"
